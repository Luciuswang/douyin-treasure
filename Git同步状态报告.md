### 方法3：手动检查命令
```bash
# 查看远程仓库
git remote -v

# 查看当前状态
git status

# 查看提交历史
git log --oneline -10

# 获取远程更新
git fetch origin

# 对比本地和远程
git status -sb

# 查看未推送的提交
git log origin/main..HEAD --oneline

# 查看未拉取的提交
git log HEAD..origin/main --oneline
```

1. **开始工作前**：务必先拉取最新代码（运行 `更新代码.bat`）
2. **完成工作后**：务必提交并推送代码（运行 `提交代码.bat`）
