# 🚀 Totofun 突突翻 - 快速启动指南

## 📋 问题解决

如果你看到 `ERR_CONNECTION_REFUSED` 错误，说明本地服务器没有运行。

---

## ✅ 解决方案

### 方法1：使用启动脚本（推荐）

1. **双击运行** `启动服务器.bat`
2. 等待服务器启动（会显示端口号）
3. 在浏览器中访问显示的地址（通常是 `http://localhost:3000` 或 `http://localhost:5000`）

### 方法2：使用命令行

```bash
# 方式1：使用npm脚本
npm run dev

# 方式2：直接使用serve
npx serve . --cors

# 方式3：指定端口
npx serve . --cors -l 3000
```

### 方法3：使用Python（如果没有Node.js）

```bash
# Python 3
python -m http.server 3000

# Python 2
python -m SimpleHTTPServer 3000
```

---

## 🌐 访问地址

服务器启动后，在浏览器中访问：

- **默认地址**: `http://localhost:3000/index.html`
- **或者**: `http://localhost:5000/index.html`（取决于serve的默认端口）
- **或者**: `http://127.0.0.1:3000/index.html`

---

## 🔍 检查服务器状态

### Windows PowerShell
```powershell
netstat -ano | findstr :3000
```

### 如果看到输出，说明服务器正在运行

---

## ⚠️ 常见问题

### 1. 端口被占用

**错误**: `Port 3000 is already in use`

**解决**:
```bash
# 使用其他端口
npx serve . --cors -l 5000
```

### 2. 找不到npm命令

**错误**: `'npm' 不是内部或外部命令`

**解决**: 需要安装Node.js
- 下载: https://nodejs.org/
- 安装后重启命令行

### 3. 防火墙阻止

**解决**: 允许本地服务器通过防火墙

## 📱 移动端测试

### 在同一网络下

1. 找到你的电脑IP地址（如：192.168.1.100）
2. 在手机浏览器访问：`http://192.168.1.100:3000/index.html`

### 查看IP地址

**Windows**:
```bash
ipconfig
# 查找 IPv4 地址
```
