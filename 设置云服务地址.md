# ⚙️ 设置云服务API地址

部署到Railway/Render后，需要在前端配置API地址。

## 方法1：在代码中设置（推荐）

1. 打开 `index.html`
2. 找到第723行左右的 `API_CONFIG` 部分
3. 修改 `CLOUD_API_URL`：

```javascript
const API_CONFIG = {
    // 设置你的Railway/Render域名
    CLOUD_API_URL: 'https://你的域名.railway.app',  // 例如：'https://totofun-server.up.railway.app'
    
    // ... 其他配置
};
```

4. 保存并推送到GitHub

## 方法2：通过浏览器控制台设置（临时测试）

1. 打开网站
2. 按F12打开开发者工具
3. 在控制台输入：

```javascript
localStorage.setItem('API_BASE_URL', 'https://你的域名.railway.app');
location.reload();
```

## 方法3：通过URL参数设置（临时）

在URL后添加参数：
```
https://luciuswang.github.io/douyin-treasure/?api=https://你的域名.railway.app
```

## 验证设置

1. 打开浏览器控制台（F12）
2. 查看Network标签
3. 尝试注册/登录
4. 检查请求URL是否为你的云服务地址

## 常见问题

**Q: 如何知道我的Railway域名？**
A: Railway项目页面 → Settings → Generate Domain → 复制域名

**Q: 设置后还是连接失败？**
A: 检查：
- 域名是否正确（包含https://）
- Railway服务是否运行（访问 /health 端点）
- CORS设置是否正确
